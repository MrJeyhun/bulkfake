##  Getting Started

###  Installation

1. Clone the bulkfake repository:

```sh
git clone https://github.com/MrJeyhun/bulkfake
```

2. Install the dependencies:

```sh
npm install
```

###  Running bulkfake

Use the following command to run bulkfake:

```sh
npm run dev
```
